# Tombola Cantata — Sito ufficiale (GitHub Pages)

Questo repository contiene i file del sito di **Tombola Cantata** (HTML statico) pronto per GitHub Pages.

## Cosa trovi
- `index.html` — pagina principale
- `img/` — logo, favicon e screenshot reali
- `Condizioni_Vendita_Tombola_Cantata.pdf` — condizioni di vendita (PDF)
- `.nojekyll` — evita problemi con GitHub Pages

## Pubblicazione su GitHub Pages
1. Carica questi file nel repository (branch `main`).
2. Vai su **Settings → Pages**.
3. Seleziona:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Salva. Dopo poco avrai l'URL pubblico del sito.

## Modifiche rapide
- Numero WhatsApp + messaggio precompilato: cerca `wa.me` dentro `index.html`.
- Prezzi: cerca la sezione **Prezzi e Licenze** dentro `index.html`.
